# Apothematic
Apothematic is a Python file which will give you all details about a 2D or 3D shape of your choice. This includes prisms, cones, cylinders, and pyramids which can have circular, rectangular, square, pentagonal, hexagonal, trapezoidal, and regular polygon bases. This program will return LA, TA, V, and Slant Height (if applicable) of 3D shapes and all relevant details of their bases.
## Dependencies
There are no dependencies for Apothematic.
